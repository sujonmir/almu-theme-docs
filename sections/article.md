# Article (Single Blog Post)

The single-blog-post template. Renders the article title, author, dates, content, social share buttons, related posts, and comments.

## Features

- **Reading progress bar** — fixed at the top, smoothly tracks how far through the article the reader is
- **Reading time** — "X min read" before the title (computed from word count, ~200 wpm)
- **Author + publish date + last edited date** — in a single row under the title
- **Hero image** — full-width image above the title (height configurable)
- **Sidebar** — optional left or right sidebar with recent posts and tag list
- **Social share** — 7 platforms (Facebook, X, Pinterest, LinkedIn, WhatsApp, Email, Copy link) each with brand-color hover
- **Related articles** — automatically populated by shared tags
- **Comments** — Shopify-native comment system with moderation support

## Settings

### General

| Setting | Description |
| --- | --- |
| **Show breadcrumbs** | Home → Blog → Article |
| **Breadcrumb separator** | Slash, chevron, bullet, double chevron, or arrow |
| **Show author / publish date / last edited** | Toggle each piece of the meta row |
| **Last edited label** | Text before the edited date — bold (e.g., "Last Edited:") |
| **Show article tags** | Tags under the title |

### Reading progress bar

| Setting | Description |
| --- | --- |
| **Show progress bar** | Master toggle |
| **Progress bar height** | 2–10 px |
| **Progress bar color** | Defaults to the accent color |

### Reading time

| Setting | Description |
| --- | --- |
| **Show reading time** | Before the title |
| **Reading time label** | Text suffix (defaults to "min read") |
| **Color, font size (desktop / mobile)** | Typography |

### Typography

Title size (desktop / mobile), title weight, body text size (desktop / mobile), meta size (desktop / mobile).

### Spacing

Padding top / bottom (desktop / mobile), container side padding (desktop / mobile).

### Featured image

Show / hide, image height.

### Social sharing

Toggle each of the 7 platforms independently. Each has its own brand color hover (Facebook #1877F2, X #000, Pinterest #E60023, LinkedIn #0A66C2, WhatsApp #25D366, Email #EA4335, Copy link uses accent). The Copy button turns green for 1.5 seconds when the link is copied.

### Sidebar

Enable, position (left/right), recent posts count, tag list heading.

### Related articles

Heading, count (2–6). Articles are matched by shared tags.

### Comments

Comment form input style: **Full Border** (legend label) or **Border Bottom** (floating label).

### Colors

Heading, body text, meta info, accent, comment text, related-card background.
