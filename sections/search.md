# Search Results

The `/search` template. Renders results matching the query: products, articles, and pages.

## Features

- Faceted filtering (same UI as the collection page) — by availability, price, and any custom Shopify filter
- "Did you mean…" suggestion when no results match the literal query
- Result counts per type (Products: N, Articles: M, Pages: K)
- Paginated grid

## Settings

| Setting | Description |
| --- | --- |
| **Results per page** | Pagination |
| **Show product count** | Display total under the search bar |
| **Show suggestions** | Show "Did you mean" + popular queries when zero results |
| **Empty-state heading / body** | Customise the "No results" message |
