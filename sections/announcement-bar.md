# Announcement Bar

Sticky banner above the header. Useful for shipping promotions, flash sale countdowns, or service announcements.

## Settings

| Setting | Description |
| --- | --- |
| **Enable** | Master toggle |
| **Text** | The announcement text (supports a link via the URL field) |
| **Link URL** | Wraps the text in an anchor |
| **Background, text color** | Brand colors |
| **Padding** | Vertical padding |
| **Show on mobile** | Hide on narrow screens if it crowds the header |
| **Marquee** | Optional scrolling text effect |
