# Blog (Article Listing)

The blog index template. Renders a grid or list of article cards.

## Layout

- **Grid** (default) — N columns
- **List** — image-left, content-right

## Card content

Each article card shows: image, tags (optional), reading time (optional, "X min read" before the title), title, author, publish date, excerpt, "Read more" link.

## Customisation

Heading size + weight (desktop / mobile), card padding (desktop / mobile), card title size + weight (desktop / mobile), excerpt size (desktop / mobile), meta size (desktop / mobile), reading time size + color (desktop / mobile), card border radius, gap between cards (desktop / mobile), column count (desktop / tablet).

## Colors

Section background, heading, body text, accent (tags + links), reading time, card background, card border.
