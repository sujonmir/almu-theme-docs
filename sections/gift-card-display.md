# Gift Card Display

The `/gift_cards/<code>` template. Shown when a customer receives a gift card and clicks the link in their email.

## Features

- Logo or shop name display
- Gift-card image (custom upload or shop logo fallback)
- Visible code + copy-to-clipboard button
- "Add to Apple Wallet" button (when the order qualifies)
- Print-friendly layout (prints the card on a single sheet)

## Settings

Logo + visual image upload, show shop name fallback, layout, all colors. The print stylesheet auto-applies when the user prints.
