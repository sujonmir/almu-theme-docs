# Password Page

Shown when your store is password-protected (Shopify admin → **Settings → Preferences → Password protection**). Fully customisable.

## What you can configure

- Logo (image + width desktop/mobile + space below)
- Header text (size desktop/mobile, weight, line height, letter spacing, transform, margin)
- Paragraph (size desktop/mobile, line height, opacity, margin)
- Password label (screen-reader only), placeholder, button text
- Layout: content alignment (L/C/R), vertical alignment (top/center/bottom — desktop and mobile separately), container max width (desktop in px + mobile in %), container padding (desktop / mobile)
- Input & button: padding (desktop / mobile), font size (desktop / mobile), border width, border radius, font weight, letter spacing, text transform
- Colors: background, text, input bg/text/border, button bg/text + hover variants
- Background image with overlay opacity
- Countdown timer (optional) — heading + 4 segments (days/hours/minutes/seconds) with full typography and styling controls

## When countdown is useful

If you're launching soon, set the countdown end date (YYYY-MM-DD) to your launch and the timer auto-displays in days/hours/minutes/seconds, updating every second.

## Mobile

Every dimension has a separate mobile value. Test on a 375 px viewport before going live.
