# Page Template

Used by static pages (About, FAQ, Shipping, Size Guide, etc.). Highly customisable: hero, breadcrumbs, title, subtitle, meta row (author, date, reading time), featured image, content, tags, share buttons, newsletter CTA, and reading progress bar.

## Width

| Option | Behaviour |
| --- | --- |
| **Default** | Follows the theme-wide page width (Theme settings → Layout) |
| **Full width** | Spans the entire viewport (1 / -1 grid column) |

## Reading progress bar

A fixed bar at the top of the viewport that fills as the user scrolls through the page content. Toggle, height (2–10 px), and color (defaults to accent) are configurable.

## Hero banner (optional)

A full-bleed header above the page content. Configure:

- Background image (with overlay color + opacity)
- Background color (if no image)
- Hero text color
- Padding (vertical), min height, border radius, margin below
- Text alignment
- Title font size and subtitle font size

If hero is enabled, the title + subtitle + breadcrumbs can render inside the hero instead of above the content.

## Header (when hero is disabled)

- Title (with full typography controls: size desktop/mobile, weight, line height, letter spacing, text transform)
- Subtitle (size desktop/mobile, color)
- Meta row (author + avatar, publish date, reading time)

## Featured image

Optional image inside the content area with optional caption.

## Content

The page's rich-text content from the Shopify admin. Almu styles all rte elements (h1–h6, p, ul/ol, blockquote, img, a, table, code).

## Tags

A comma-separated list rendered as styled pills under the content.

## Share buttons

7 platforms with brand-color hover (Facebook #1877F2, X #000, Pinterest #E60023, LinkedIn #0A66C2, WhatsApp #25D366, Email #EA4335, Copy link uses accent). Each is independently toggleable.

## Newsletter CTA

Optional newsletter sign-up block at the bottom of the page (separate from the global newsletter popup).

## Colors

Page background, card background, text, secondary text, title, subtitle, border, accent, accent contrast.

## Typography

Title (desktop / mobile font size, weight, line height, letter spacing, text transform), subtitle (desktop / mobile size, color), body text (desktop / mobile size, line height), meta (size, color).

## Spacing

Padding top / bottom (desktop / mobile), container padding (desktop / mobile), container border radius, optional drop shadow.
