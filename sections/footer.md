# Footer

Site-wide footer. Add multiple **menu columns** (link list blocks) and configure newsletter, payment icons, and social links.

## Blocks

- **Link list** — a column with a heading + Shopify nav menu
- **Newsletter** — email sign-up block
- **Payment icons** — inline payment-method icons
- **Social** — social media icon row (URLs come from theme settings)
- **Text** — free-text column (e.g., contact info, copyright)

## Settings

| Setting | Description |
| --- | --- |
| **Background, text, heading, link color** | Brand colors |
| **Border top color / width** | Top border above the footer |
| **Padding (desktop / mobile)** | Vertical padding |
| **Show copyright** | Copyright row below the columns |
| **Show locale selector** | If you have multiple Shopify locales enabled |
| **Show currency selector** | If you have Shopify Markets enabled |
