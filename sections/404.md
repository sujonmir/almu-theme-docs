# 404 — Not Found

Rendered when a URL doesn't match any product, collection, page, or article.

## Settings

| Setting | Description |
| --- | --- |
| **Show search bar** | Renders the predictive search overlay trigger |
| **Blur effect** | Background glass/blur effect |
| **Heading text** | Default: "Page not found" |
| **Subtext** | Friendly explanation |
| **Button text + URL** | CTA back to a sensible place (homepage, all collections) |
| **Background color, text color** | Brand styling |

The page automatically sends a `noindex` meta tag so search engines don't index 404s.
