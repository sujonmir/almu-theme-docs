## What to include in a support email

The more of these you can include, the faster I can help:

1. **Your store URL** — even better if it's a development store I can preview
2. **Theme version** — find under **Theme settings → Theme info** (lower-left in the theme editor) or check the `theme_info` block in `config/settings_schema.json`
3. **What you were doing when the issue happened**
4. **What you expected to happen**
5. **What actually happened** — screenshots + the URL where the bug occurs
6. **Browser + OS + device** if it's a display issue (e.g., "Chrome 121 on iPhone 14 Pro")

## Response times

- **Weekdays:** ~24 hours
- **Weekends:** ~48 hours

Theme Store buyers get priority on urgent issues (storefront broken, can't save settings, etc.) over feature requests.

## What I can help with

- Bugs in any of Almu's sections or snippets
- Theme settings that aren't doing what you expect
- Compatibility with Shopify built-in apps and well-known third-party apps
- Customisation pointers — telling you which section / snippet / setting to touch
- Performance / Lighthouse debugging

## What's out of scope

- Building custom features for your specific store (the license is per-store but doesn't include bespoke dev work)
- Debugging third-party apps that don't follow Shopify's theme app extension API
- Customising checkout (Shopify owns checkout; theme can't modify it directly)

For larger custom work, I can refer you to a Shopify Expert.
