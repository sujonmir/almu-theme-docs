## What's in Almu

Almu is built around three principles: **performance, accessibility, and granular customisation**. Every section ships with desktop + mobile typography controls, the cart drawer keyboard-traps focus for screen-reader users, and zero render-blocking third-party scripts are loaded from the storefront.

| | |
| --- | --- |
| **Sections** | 50 ready-to-use sections covering product, collection, cart, blog, marketing, and utility layouts |
| **Languages** | 11 storefront locales (en, ar, bn, de, es, fr, hi, it, ja, pt-BR, zh-CN) |
| **Performance** | Critical CSS inlined, base CSS async-loaded, JS deferred, lazy-loaded images, LCP-tuned hero images |
| **Accessibility** | WCAG 2.1 AA — skip link, focus styles, ARIA live regions, keyboard trap in modals, mega-menu arrow nav |
| **SEO** | Organization, BreadcrumbList, and Product JSON-LD; canonical URLs, OG/Twitter tags, meta description |

## Getting started

1. [Install the theme]({{ "/installation/" | relative_url }}) on your Shopify store.
2. Configure [global theme settings]({{ "/theme-settings/" | relative_url }}) — colors, typography, cart drawer.
3. Browse the [sections reference]({{ "/sections/" | relative_url }}) to learn what each section does and how to configure it.
4. Read the [customisation guide]({{ "/customization/" | relative_url }}) for cart drawer, newsletter popup, back-to-top, etc.
5. Optional: review [performance]({{ "/performance/" | relative_url }}) and [accessibility]({{ "/accessibility/" | relative_url }}) features.
