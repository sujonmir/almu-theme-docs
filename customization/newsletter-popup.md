# Customising the Newsletter Popup

The newsletter popup is a separate section from the inline "Newsletter" block in the footer / page template.

## Recommended setup

1. **Trigger** — start with "Delay: 30 s" + "Show once per session". This is the least intrusive default.
2. **Copy** — keep it short. Heading + 1 sentence + clear CTA.
3. **Discount code** — generate a discount in **Shopify admin → Discounts**, then enter the code in the popup settings. It's revealed only after successful sign-up.
4. **Visual** — pair a left-side image with right-side form for the strongest conversion.
5. **Hide on mobile** — if your mobile traffic is high and the popup feels too aggressive, hide it there.

## What gets stored

Email addresses captured via the popup are added to your Shopify customer list with the tag `newsletter` (configurable in the section settings).
