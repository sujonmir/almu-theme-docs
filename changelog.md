# Changelog

All notable changes to Almu.

## 1.0.0 — 2026-05-27

Initial public release. Submitted to the Shopify Theme Store.

### Features

- 50 sections covering header, footer, product, collection, cart, blog, article, page, marketing, and utility layouts
- 38 snippets (cart drawer, product card, share buttons, icon system, etc.)
- 11 storefront + 11 schema locales (English default + Arabic, Bengali, German, Spanish, French, Hindi, Italian, Japanese, Portuguese, Simplified Chinese)
- WCAG 2.1 AA accessibility (skip link, focus styles, ARIA live regions, keyboard trap, mega-menu arrow nav)
- SEO: Organization, BreadcrumbList, and Product JSON-LD; canonical URLs, OG/Twitter Card, meta description
- Self-hosted Swiper (no external CDN dependencies)
- Critical CSS inlined; base CSS async-loaded; JS deferred; LCP-eager primary product image
- Cart drawer with shipping-note threshold, order note, payment icons, line-item ARIA live
- Configurable back-to-top button (position, shape, size, threshold, smooth scroll, mobile toggle)
- Newsletter popup with delay/scroll/exit triggers and discount-code reveal
- Reading progress bar on page + article templates
- Reading time ("X min read") on article cards and article pages
- Author + publish date + last-edited date row on article templates
- 7-platform social share (Facebook, X, Pinterest, LinkedIn, WhatsApp, Email, Copy link) with brand-color hover
