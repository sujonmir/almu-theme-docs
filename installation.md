# Installing Almu

1. TOC
{:toc}

---

## 1. Purchase and add the theme

1. Go to the [Almu theme listing on the Shopify Theme Store](https://themes.shopify.com/).
2. Click **Try theme** to demo it on your store before paying, or **Buy theme** to purchase directly.
3. After purchase, Shopify will automatically add Almu to your store's **Online Store → Themes** library.

## 2. Publish the theme

You can preview Almu without affecting your live storefront:

1. In your Shopify admin go to **Online Store → Themes**.
2. Find Almu under **Theme library**.
3. Click **Actions → Preview** to see Almu running against your real products and collections — this doesn't change your live theme.
4. When ready, click **Actions → Publish** to make Almu your live theme.

> **Tip:** Before publishing, duplicate your current live theme as a backup (**Actions → Duplicate**).

## 3. First-time setup checklist

After publishing, walk through these steps to get a polished storefront fast:

- [ ] **Logo** — upload your logo via the theme editor → **Theme settings → Logo**
- [ ] **Brand colors** — set primary, accent, and text colors in **Theme settings → Colors**
- [ ] **Typography** — pick your heading and body fonts in **Theme settings → Typography**
- [ ] **Cart drawer** — enable and configure under **Theme settings → Cart**
- [ ] **Header** — choose your menu, logo placement, and search behaviour in the header section
- [ ] **Footer** — set up your footer menu, payment icons, and newsletter sign-up
- [ ] **Homepage sections** — drag in the sections you want from the section panel
- [ ] **Announcement bar** — enable if you want a sticky banner at the top
- [ ] **Back-to-top button** — enable in **Theme settings → Back to top**

## 4. Required Shopify settings

A few things to set up in your Shopify admin (not the theme editor):

- **Policies** — Shopify → **Settings → Policies** — fill in privacy, refund, shipping, and terms. Almu's footer auto-links to whichever policies you publish.
- **Currency** — Shopify → **Settings → Store details → Store currency**. Almu honours Shopify's currency conversion.
- **Locales** — Shopify → **Settings → Languages**. Almu ships with translations in 11 languages (English default + 10 published locales). Enable the languages you sell in.
- **Markets** (optional) — if you sell internationally, configure Shopify Markets and Almu will pick up the market's currency and locale.

## 5. Troubleshooting

| Problem | Fix |
| --- | --- |
| "Theme couldn't be published" | Refresh, try again. If it persists, your theme library may be at Shopify's free-plan limit (3 themes). Delete an unused one. |
| Mega menu doesn't show sub-items | The header section's menu setting must point to a Shopify navigation that has nested links. |
| Cart drawer opens empty | Confirm **Theme settings → Cart → Cart type** is set to **Drawer**, not **Page**. |
| Search overlay doesn't trigger | The header's "Show search" toggle must be on. |
| Newsletter popup never appears | Check **Newsletter popup** section's "Show on" timing — defaults can be very delayed. |
